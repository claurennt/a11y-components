# A11y Components

Production-ready accessibility examples for common UI components. Learn how to build accessible versions of components you use every day.

## What This Is

Real-world examples showing how to implement common components with proper:

- Keyboard navigation (arrows, tab, escape)

- Screen reader support (ARIA labels, live regions)

- Focus management (logical order, visible indicators)

- WCAG compliance (following 2.1 AA guidelines)
